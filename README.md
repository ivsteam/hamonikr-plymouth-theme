Kevin Kim <root@hamonikr.org>

# Installation

```
$ wget https://github.com/ivsteam/hamonikr-plymouth-theme/archive/1.0.tar.gz
$ tar -xvzf 1.0.tar.gz
$ cd hamonikr-plymouth-theme
$ sudo ./install.sh
```
